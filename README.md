# Setup

Expects node and npm to be already installed.
```bash
npm install -g bower
npm install -g bowcat
npm install
bower install
bowcat -o assets -m
```

# Run after installing/uninstalling bower components

```bash
bowcat -o assets -m
```

# Run webserver

## Windows
```bash
run.bat
```

## Linux/Unix-like
```bash
chmod +x run
./run
```
