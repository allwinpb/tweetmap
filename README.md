# Setup

Expects node and npm to be already installed.
```bash
npm install -g bower
npm install -g bowcat
npm install
bower install
recompile (Windows) OR sh recompile.bat (~Linux)
```

# Run after change to bower_components

```bash
recompile (Windows)
sh recompile.bat (~Linux)
```

# Run webserver

```bash
run (Windows)
sh run (~Linux)
